# File Organizer Automation Script

This is a Python-based automation tool that cleans up a messy directory by sorting files into relevant folders based on their extensions.

## Features
- **User Input Support**: Allows the user to specify any folder path.
- **Error & Exception Handling**: Validates paths and handles permission errors smoothly.
- **Log Generation**: Creates an `automation_log.txt` keeping track of every operation.

## How to Run
1. Open terminal/command prompt.
2. Run the script using: `python main.py`
3. Enter the absolute folder path when prompted.
